MongoDB Interactions
 
Goto mongodb bin folder and open cmd and type mongod. This will start the mongodb.
Open another cmd and type mongo. This will enable us to perform actions like insert, update, create.
 
Creating a data table
                use chartlist
 
 
Find records
                db.chartlist.find().pretty();
 
 
 
To run this project:
 
open the command prompt of this project and do the following.
1- npm install
2- node server.js
3- open http://localhost:3000 in the browser
